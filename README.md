# Easy Generator Task

This is a full-stack application consisting of a React frontend and a NestJS backend.

## Deployed Application

- **Frontend**: [https://easy-generator-fe.vercel.app/](https://easy-generator-fe.vercel.app/)
- **Backend**: [https://easy-generator-be.vercel.app/](https://easy-generator-be.vercel.app/)

## Project Structure

- **frontend**: React application built with Vite.
- **backend**: REST API built with NestJS.

## Quick Start

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn
- MongoDB instance (for the backend)

### Running the Application

1.  **Backend**:
    ```bash
    cd backend
    npm install
    # Set up .env file (see backend/README.md)
    npm run start:dev
    ```

2.  **Frontend**:
    ```bash
    cd frontend
    npm install
    # Set up .env file (see frontend/README.md)
    npm run dev
    ```

For detailed instructions, please refer to the `README.md` files in the `frontend` and `backend` directories.
